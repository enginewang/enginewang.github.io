发表于 2018-10-06 | 分类于 Tech | | 阅读次数:

Xpath Selector


Expression Description
nodename select nodes with the name
. select the current node
.. select the parant of current node
/ select from root node
// select from this node no matter where they are
@ select attributes
* match any elem node
@* match any attribute node


Node predicate Description
[num] -
[last()] -
[position()] -
[@id=””] -
[@class=””] -



CSS Selector


Selector Example Description
.class .nav select elems whose class is “nav”
#id #fruit select elems whose id is “fruit”
* * select all elems
elem p select <p> elems
elem, elem div, p select <div> elems and <p> elems
elem elem div p select <p> elems inside <div> elems
elem>elem div > p select <p> elems whose parent is <div>
[attribute] [target] select elems with a target attrtbute
[attr~=v] [title~=flower] whose title contains flower
[attr^=v] a[href^=”https”] whose href begins with “https”
[attr$=v] a[href$=”com”] whose href ends with “com”
[attr*=v] a[href*=”engine”] whose href contains “engine”
