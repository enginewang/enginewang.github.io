发表于 2018-09-30 | 分类于 Tech | | 阅读次数:

What is numpy?  Here is the definition of numpy library in wikipedia:

NumPy/ˈnʌmpaɪ/ is a library for Python, adding support for large, multi-dimensional arrays and matrices, along with a large collection of high-level mathematical functions to operate on these arrays.

In a word, numpy can make it more convenient for us to operate array and matrix in Python, with high efficiency.
